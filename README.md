# bunpo2nippo

Extract tweets from your slack channel

WIP

```
npm install bunpo2nippo -g
```

# Setting

```
{
  "token":"your-slack-api-token",
  "userName":"mizchi",
  "channelName":"status_mizchi"
}
```

# run

```
$ bunpo2nippo
pm 05:43 なんとなく全員の分報読んでみたけど全部読むもんじゃないな
pm 06:23 分報から自動で日報生成する奴作ってる
pm 08:23 めしった
```
